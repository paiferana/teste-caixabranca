Class User

Package: login

Overview

This class is responsible for handling database connections and user authentication.

Methods

  • Connection conectarBD()

    Establishes a connection to the MySQL database.

    Returns: Connection - A database connection object.

  • boolean verificarUsuario(String login, String senha)

    Validates user credentials (login and password).

    Parameters:

    • login: User's login name.
    • senha: User's password.

    Returns: boolean - True if the credentials are valid, false otherwise.

Fields

  • String nome - Stores the name of the user.
  • boolean result - Indicates whether the login was successful.
